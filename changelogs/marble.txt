This is Android 16 with September 2025 security patches

Notes:
==============================
- Install a HyperOS 2 FW according to your region (https://xmfirmwareupdater.com/firmware/marble)
- LineageOS-based source. Signed build.
- Use OFRP-R11.1_7-Beta-marble.zip as recovery.
- Always dirty flash at your own risk. If your device bootloops, revert to an older build that boots.
- Install this (https://sourceforge.net/projects/evolution-x/files/marble/16/Evo-11.x-5.10.240-KSU-Next-v1.1.1-susfs-1.5.10.zip/download) for root.
- Includes POCO camera and Dolby Atmos.
- Credits to Arian16k, GR, Chaitanya, Adrianyyyy.

Device changes:
==============================
Use 8Gb dalvik heap size config
libinit: Drop 16gb dalvik heap config
parts: Drop per-app refresh rate (moved to Settings > Display > Refresh rate)

Settings: Ambient Music Ticker - Allow to pulse on new tracks

SystemUI: Show icon of package associated with Toast
SystemUI: Utilities: Update for media control
Update translations
